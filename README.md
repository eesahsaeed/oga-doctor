# OgaDoctor

React + Express + Vite fullstack boilerplate.

## Structure

- `client/` - Vite React frontend
- `server/` - Express backend (migrated from `ogadoctor-backend`)

## Run

```bash
npm run dev
```

This starts:
- Vite client dev server
- Express backend server

## Other scripts

```bash
npm run dev:client
npm run dev:server
npm run build
npm run preview
npm run test:server
```
